Projects

Computer Graphics - Island Temple Application
  • Rendered a Temple on an Island Terrain using C++ with OpenGL, GLUT & the ANGEL library.
  • Generated terrain from a height map with water simulation around it.
  • Implemented a flying camera for first person exploration.
Test Case Suite Application
  • Developed Test Case Suite Application in C# with team as team leader while following Software Engineering principles (Agile methodology).
  • Implemented parsing of C++ source code & generation of suggested test cases & test drivers to the user.
  • Coordinated development process: design, documentation, & implementation activities.
Where’s Waldo Browser Game
  • Developed web-based game where player must find randomly placed waldo character hidden somewhere on an image.
  • Used HTML, CSS, JavaScript & PHP for development as well as Mongo to store random generated waldo’s location on an image.
  • Implemented timer to record player detection speed & blur image feature to make sure player starts fairly.
Donation Tracker Application
  • Developed Java application to manage a MySQL database of donations with a team.
  • Managed the GitHub repository for the project by organizing project files, reviewing, & accepting pull requests from team members.
  • Implemented donation report generation on data queried utilizing open source Jasper Reports library.
Press Your Luck Trivia Board Game
  • Reinvented Press Your Luck game as Windows C# application with partner.
  • Implemented questioning & validation of answers, multiplayer, spinning gameboard wheel, & game flow.
  • Designed GUI, layout of game board & board space graphics.
Parallel Fast Fourier Transform (Cooley-Tukey Algorithm)
  • Implemented Fast Fourier Transform (FFT) radix-2 DIT case (C & Cuda).
  • Improved efficiency of the FFT with parallel computing using Cuda.
  • Code compiled & executed on Texas Advanced Computing Center’s Maverick with 132 NVIDIA Tesla K40 GPUs.

About Me

I'm an aspiring software developer, who hails from a small island in the Caribbean called Antigua. From a young age, technology has been a keen interest of mine and that curioustiy led me to set out on the path of becoming an industry software professional. I'm passionate about developing software that adds to the comfort, experience, and satisfaction of the customer. I strongly believe that the future of technology and human-computer interaction will be paved by High Performance Computing, Computer Vision and Artificial Intelligence.
